1. Escalate hard decisions with the advisor tool 用 advisor 工具升級困難決策

advisor 工具讓 Claude 在任務關鍵時刻（決定方案前, 卡在重複出現的錯誤時, 宣告任務完成前）諮詢第二個, 通常更強的模型.advisor 收到完整對話並回傳 Claude 會採納的建議, 為實驗性功能, 僅支援 Anthropic API.

三種設定方式

選擇方式 :
/advisor 指令
advisorModel 設定
--advisor 旗標

互動選擇 advisor 模型並存為預設, session 間持續生效.

在 settings 檔設定持久預設值, 不需開啟 session 即可配置.

為單次 session 設定, 優先於已儲存的 advisorModel.

主模型可接受的 advisor
Haiku 4.5Fable, Opus, Sonnet
SonnetFable, Opus, 同版或更新的 Sonnet
OpusFable, 同版或更高版本的 Opus
Fable 5僅 Fable

Claude 只在決策點呼叫 advisor 而非每回合, 通常比全程使用更強模型划算./advisor off 可停用, CLAUDE_CODE_DISABLE_ADVISOR_TOOL=1 可完全關閉此功能.

2. Champion kit 內部推廣者手冊

給已經在使用 Claude Code 並想協助團隊採用的工程師.採用很少來自推廣公告, 而是因為某人把工具用得好, 公開分享, 並讓其他人容易跟上.

分享你的發現
把提示詞, 截圖, 小勝利貼到團隊已經在看的地方
成為大家會問的人
回答時附上實際用的提示詞而非解釋
擴大圈子
建立少量輕量, 可持續的習慣
每週投入
分享約 15 分鐘, 答問約 20 分鐘, 分享串約 5 分鐘

30 天推廣時程表

第一週 · 播種
第二週 · 節奏
第三週 · 結對
第四週 · 交棒

建立頻道, 置頂 Quickstart, 貼出兩三個自己的範例與提示詞.

啟動每週分享串, 公開回答每個問題, 分享一個自訂 skill 或 CLAUDE.md 片段.

提供兩三次短時間結對, 把常見問答整理成置頂 FAQ.

找出下一位推廣者, 向主管或管理者摘要成效.

3. Communications kit 組織溝通套件

給要在工程組織內推出 Claude Code 的管理者與工程主管, 提供可直接複製使用的發布公告, 逐週提示技巧宣傳文案, 常見問答一行回覆.

發布前檢查清單

頻道建立 #claude-code 頻道並連結到公告中
安裝測試在自己環境測試過安裝指令, 提早發現代理或防火牆問題
安全連結準備好資料處理與安全性文件連結
第一個任務選定一個具體, 來自自己程式碼庫的範例
高層贊助具名發送公告的採用率高於管理者發送
宣傳文案主題

模型選擇, /init 產生 CLAUDE.md, @ 檔案引用, 權限模式, checkpointing 與 /rewind, MCP 連接器, skills, hooks, 螢幕截圖輸入, git 工作流程, plugins, 每則文案遵循相同結構, 吸引點, 效益, 立即可試的動作, 文件連結.

4. Prompt library 提示詞庫

依開發生命週期階段與角色標記的可直接複製貼上提示詞集合, 涵蓋探索, 規劃, 實作, 測試, 審查, 部署各階段.

# 探索階段範例
"give me an overview of this codebase: architecture, key directories, and how the pieces connect"
"explain what {path} does and how data flows through it"
"where do we {behavior}?"
"what would break if I deleted {target}?"

每個提示詞條目包含可代換的占位符與後續延伸閱讀連結, 定位為降低撰寫有效提示詞門檻的參考起點, 適合分享頻道或內部 wiki 供團隊直接取用.
